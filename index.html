QUIZZIPEDIA


QuizziPedia è un'applicazione web per la gestione di questionari. Il progetto è stato proposto dall'azienda Zucchetti S.p.a. per il corso di Ingegneria del Software ed è stato realizzato in collaborazione con altri sei studenti. Le principali tecnologie utilizzate sono: Angular.js, Node.js, MongoDB, HTML5, CSS3.

  • GitHub: Link al progetto
  • Data di realizzazione: Novembre 2015 - Giugno 2016

PCD-ACTORS


PCD-Actors è un framework sviluppato in Java che implementa un minimale modello ad attori che prevede uno scambio di messaggi in maniera concorrente. Il progetto è stato sviluppato individualmente per il corso di Programmazione Concorrente e Distribuita.

  • GitHub: Link al progetto
  • Date: Dicembre 2015 - Gennaio 2016

LINQEDIN


LinQedIn è un social network minimale per l’amministrazione ed utilizzo, tramite interfaccia utente grafica, di un piccolo database di contatti professionali ispirato a LinkedIn. Il progetto, individuale, è stato sviluppato in C++ e librerie Qt, con salvataggio fisico dei dati in XML, per il corso di Programmazione ad Oggetti

  • GitHub: Link al progetto
  • Date: Gennaio 2015 - Giugno 2015

PALESTRADB


PalestraDB è un database relazionale che, accompagnato da un applicativo web, permette la gestione di una palestra. Il progetto è stato sviluppato per il corso di Base di Dati ed è stato realizzato in collaborazione con un altro studente. Le principali tecnologie utilizzate sono: HTML, CSS, PHP, SQL e MySQL.

  • GitHub: Link al progetto
  • Date: Luglio 2015 - Settembre 2015
